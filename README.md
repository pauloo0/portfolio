# Developer Portfolio 2026

A modern, high-performance developer portfolio built with React, TypeScript and Tailwind CSS.

## 🛠️ Tech Stack

- Framework: React v19
- Language: TypeScript
- Styling: Tailwind CSS v4
- Icons: Lucide React
- Deployment: VPS (Ubuntu + Caddy)

## 🎨 Design System

- Color Palette:
  - Primary: #354F52
  - Secondary: #52796F
  - Accent: #84A98C
  - Background: #2F3E46
  - Text: #EEF7E8
- Typography:
  - Heading: Lora
  - Body: Open Sans
